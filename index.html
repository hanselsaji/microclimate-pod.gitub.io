Project Overview

A comprehensive IoT solution for microclimate monitoring and analysis

What is the Smart Microclimate Pod?

An advanced IoT-enabled smart ecosystem that automatically detects plant species and precisely adjusts climate conditions (temperature, humidity, light, CO₂) to their optimal levels. Combining computer vision, automation, AI recommendations, and voice control, it creates a perfectly balanced micro-environment tailored to each plant's specific needs while also enhancing human wellness through natural aromatherapy.

Core Features

  • Plant-based humidification with natural essential oil diffusion
  • Precision watering based on real-time soil moisture analysis
  • Google Assistant and Alexa voice control integration
  • Real-time monitoring via responsive web/mobile dashboard
  • Automatic plant detection and climate profile adjustment

Tech Specs

  • Raspberry Pi 4 Controller
  • DHT22 + Soil Moisture Sensors
  • 5MP Plant Camera
  • 12V Pump & Fan System

System Components

The hardware and software that power the Microclimate Pod

Temperature Sensor

Measures ambient temperature with ±0.5°C accuracy across a range of -40°C to 125°C.

DS18B20 Digital Sensor

Humidity Sensor

Provides relative humidity measurements with ±2% accuracy from 20% to 80% RH.

DHT22 Digital Sensor

Light Sensor

Measures ambient light intensity with a spectral response close to the human eye.

BH1750 Digital Sensor

CO₂ Sensor

Detects carbon dioxide concentration with ±50ppm accuracy (400-5000ppm range).

MH-Z19 NDIR Sensor

Controller Unit

Raspberry Pi 4 with 4GB RAM running custom Python scripts for data collection and processing.

Raspberry Pi 4 Model B

Plant Camera

5MP autofocus camera with computer vision capabilities for plant species identification and growth monitoring.

Raspberry Pi Camera Module 3

Additional System Components

Power System

12V DC power supply with battery backup for uninterrupted operation

Ventilation System

Quiet 120mm PWM fans with temperature-controlled speed

Watering System

12V peristaltic pump with flow sensor and anti-clog design
